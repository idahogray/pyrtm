# pyrtm

Python interface for the 'Remember The Milk' API.

## installation

```
pip install pyrtm
```

## examples

https://bitbucket.org/srid/pyrtm/src/tip/src/rtm/samples/app.py?at=default

## hacking

### running tests

first set your API keys in src/rtm/tests/apikey.txt, and then:

```
pip install tox
tox -e py27
```
